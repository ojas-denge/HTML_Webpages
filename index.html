Color Scheme

Text Color can be changed using one of Body's attribute "Text".

Background color can be changed using its other attribute "Bgcolor".

Also a Background Image can be used in Webpage using "Background".
Link also have 3 different color codes:
Link-Blue
Active Link-Red
Visited Link-Purple

These color can be changed with body's attribute, "link" "alink" and "vlink".
And these are updated color codes for link,
Link-Orange
Active Link-Green
Visited Link-Yellow
